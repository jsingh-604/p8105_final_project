Importing 2019-2022 NY Health Ranking Data

  1. Import excel
  2. Select sheets: outcomes & factors subrankings, additional measure data (mainly for demographics info)
  3. Selected necessary variables from each sheet
  4. Renamed variables
  5. Merged data frames generated from each sheet

2019 Health Ranking & Demographics Df

ny_hr2019 = read_excel('grace/ny_hr19.xls', sheet = 'Outcomes & Factors SubRankings', skip = 1) %>% 
  janitor::clean_names() %>% 
  slice(-c(1)) %>% 
  select(-ends_with("z"), -fips)

ny_demo2019 = read_excel('grace/ny_hr19.xls', sheet = 'Additional Measure Data', skip = 1) %>% 
  janitor::clean_names() %>% 
  slice(-c(1)) %>% 
  select('county', 
         'household_income', 
         'number_african_american', 
         'number_american_indian_alaskan_native', 
         'number_asian', 'number_native_hawaiian_other_pacific_islander', 
         'number_hispanic', 
         'number_non_hispanic_white') %>% 
  rename(median_income = household_income, 
         n_black = number_african_american, 
         n_american_ind_al_native = number_american_indian_alaskan_native, 
         n_asian = number_asian, 
         n_nat_ha_other_pac_is = number_native_hawaiian_other_pacific_islander, 
         n_latino = number_hispanic, 
         n_white = number_non_hispanic_white)

ny_hr2019_fin = merge(x=ny_hr2019, y=ny_demo2019, by="county", all.x=TRUE)

Importing 2020 to November 2022 NY COVID-19 Data

  1. Decided to look at variables related to covid testing (i.e., # positive cases, # tests performed), covid vaccination rate, and covid-related fatalities.
  2. Imported 3 separate csv files (ny_covidtest, ny_coviddeaths, ny_covidvax)
  3. Selected variables of interest
  4. Merged 3 separate dataframes into one by date and county
  5. Converted date data type from character to date
ny_test = read_csv('grace/ny_covidtest.csv') %>% 
janitor::clean_names() %>% 
filter(geography == 'COUNTY') %>% 
select(-geography) %>% 
rename(date = test_date)

ny_death = read_csv("grace/ny_coviddeaths.csv") %>% 
janitor::clean_names() %>% 
rename(date = report_date, n_fatality = place_of_fatality) %>% 
select(-deaths_by_county_of_residence) 

ny_vax = read_csv("grace/ny_covidvax.csv") %>% 
janitor::clean_names() %>% 
rename(date = report_as_of) %>% 
select(-region, -series_complete)

ny_covid_v1 = left_join(ny_test, ny_death, by = c("date", "county"))
ny_covid_fin = left_join(ny_covid_v1, ny_vax, by = c("date", "county")) %>% 
  mutate(date = lubridate::mdy(date))
